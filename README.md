## Tech Stack

This project is a complete MERN stack project.

| Layer       | Tech                          |
|------------|-------------------------------|
| Frontend   | React, Tailwind CSS, Vite     |
| Backend    | Node.js, Express.js           |
| Database   | MongoDB with Mongoose         |
| Auth       | JWT, Cookies (HttpOnly + Secure) |
| Deployment | Render                         |
